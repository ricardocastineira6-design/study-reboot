Pequeña aplicación de apoyo al estudio. Esta versión está preparada para entregarse como un proyecto de JavaScript/React y funciona totalmente en local usando LocalStorage (no requiere cuentas ni servicios externos).

Características principales
- Notas: crear, editar, borrar y marcar como completadas.
- Registro de estado de ánimo: guardar cómo te sientes cada día con emoji y nota opcional.
- Temporizador de sesiones de estudio y estadísticas básicas.

Tecnologías
- Frontend: React 18 + TypeScript
- Build: Vite
- Estilos: CSS Modules
- Persistencia: LocalStorage (todos los datos se guardan en tu navegador)

Requisitos
- Node.js v18+ y npm

Instalación y ejecución
1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor de desarrollo:

```bash
npm run dev

```
